github上传代码

发表于 2018-04-27

创建仓库

在github个人主页创建一个新的仓库，点击按钮 “clone or download”,复制仓库地址：git@github.com:rexana3/test.git

git clone git@github.com:rexana3/test.git

打开一个地址（假设为E：/）,点击鼠标右键，选择“git bash”，
输入“git clone git@github.com:rexana3/test.git”，在E盘中会出现一个新的文件夹，名为：test。关闭命令行窗口，进入test文件夹（里面有一个.git文件夹和一个readme.md的文件）。右键再次打开“git bash”。

git add Demo.html

输入 “git add XXX”，XXX表示要上传的文件名。“git add *” 表示上传所有文件

git commit -m “update”

“update” 为备注内容

git push -u origin master
